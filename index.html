Лабораторная работа 5: Классы и модули

Тема: "Корзина покупок" (номер 15)

Открой консоль разработчика (F12) для просмотра результатов

Файлы проекта:

  • ShoppingCart.js - родительский класс
  • OnlineShoppingCart.js - дочерний класс
  • index.js - основной скрипт
  • index.html - эта страница

Что происходит в коде:

1. Созданы 2 экземпляра класса OnlineShoppingCart
2. Каждый имеет разное содержимое и адрес доставки
3. Вызываются методы show(), addItem(), applyDiscount()
4. Демонстрируется клонирование через static clone()
5. Показывается разница между copy() и clone()
